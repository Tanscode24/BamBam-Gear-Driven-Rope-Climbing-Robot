# String Crawl Bot (Gear-Driven Rope Climbing Robot)
# Project Overview
The String Crawl Bot is a rope-climbing robot designed to move along a suspended string using a dual shaft motor and a custom gear mechanism. The system converts rotational motion into a cyclic gripping action that enables the robot to crawl forward.<br>

This project was fully designed and built from scratch, combining CAD modeling, fabrication, and electronics integration.<br>

# Working Principle
A dual shaft DC motor drives a set of custom-designed gears. These gears convert rotational motion into alternating arm movement. The arms perform a cycle of grip, pull, and release, allowing the robot to move forward along the rope continuously.<br>

# Design and Fabrication
Designed gears and mechanism from scratch using Fusion 360<br>
Manufactured gears using 3D printing<br>
Built the chassis using laser-cut acrylic sheets<br><br>
Assembled and aligned all components manually
Integrated electronic components for power and control

Low torque → Optimized gear ratio
Misalignment → Refined assembly and structure
Weight issues → Used lightweight materials
